# Project Requirements

1.  **Design & Theme**:
    *   Dark mode background (`#0a0a0a`).
    *   Fonts: Inter and Playfair Display.
2.  **Preloader**:
    *   Display a counter from 0 to 100.
    *   Disappear after loading with an animation.
3.  **Navigation**:
    *   Logo "VICTOR."
    *   Menu toggle button.
    *   Full-screen overlay navigation when toggled.
    *   Links to Home, About, Experience, Work, Contact.
4.  **Hero Section**:
    *   Title "Hello, I'm Victor".
    *   Subtitle "Creative Developer & Tech Enthusiast".
    *   Links: GitHub, YouTube, Let's Talk.
    *   Avatar placeholder.
    *   Text reveal animations on load.
5.  **About Section**:
    *   "About Me" title.
    *   Descriptive text about the developer.
6.  **Experience Section**:
    *   "Experience" title.
    *   Timeline with at least 3 entries (Senior Developer, Full Stack Engineer, Junior Developer).
7.  **Work Section**:
    *   "Selected Work" title.
    *   Project cards (GitHub Portfolio, Project Alpha).
8.  **Contact Section**:
    *   "Have an idea?" title.
    *   Email link (`contact@victor.com`).
    *   Social links.
9.  **Interactivity**:
    *   Custom cursor (dot and outline) following mouse.
    *   Smooth scrolling (Lenis).
    *   Parallax effect on hero image/projects.
    *   Scroll-triggered animations for sections.
